# Betsaleel Mukuba — Professional Portfolio

**Live URL:** [Deploy to get your URL]

## Features
- Single-page design with smooth scroll navigation
- Particle canvas hero background
- Animated typing effect (rotating titles)
- Scroll-triggered animations & reveal effects
- Animated skill progress bars
- Project filtering by category
- Dark / Light theme toggle
- Contact form with success feedback
- Blog / Articles section
- Stats counters
- Scroll progress indicator
- Back to top button
- Preloader animation
- **Full Admin Panel** (accessible from footer)
- Fully responsive — tested down to 260px screens

## Admin Panel
Access: Click **"Admin"** in the footer
Default password: `admin2026`

Manage:
- Name, bio, headlines, titles
- Profile photo & about photo
- CV/Resume file
- Contact info & social links
- Dark/Light theme
- Admin password

## Stack
- HTML5, CSS3, Vanilla JavaScript
- Font Awesome 6 (icons)
- Syne + DM Sans (Google Fonts)

## Deployment
See `DEPLOYMENT.md` for full step-by-step Vercel deployment instructions.

## Quick Start
1. Open `index.html` in a browser — works locally with no build step
2. Customize via the Admin Panel or edit `js/main.js` CONFIG defaults
3. Deploy to Vercel (see DEPLOYMENT.md)
